Title: Scheduling System
Purpose: A desktop Java application to schedule and modify appointments with data stored and accessed in a MySQL database.  This application handles time zones and can be used in a variety of organizations located in different areas around the world.
Author: Kush Mirchandani
Contact: kmirch1@wgu.edu
Application Version: 1.0
Date: 2/14/2023

IDE: IntelliJ IDEA 2022.2.2 (Ultimate Edition)
JDK Version: 17.0.4
JavaFX version: 17.0.2

How to run the program:
    Launch the application.
    Login with one of the following valid username/password combinations: test/test, admin/admin, kush/password.

The additional report on the reports menu allows the user to select a customer from a combo box and view the number of appointments scheduled for that customer (count), and displays the Appointment ID, Title, Customer ID, Contact ID, Description, Location, Start and End of each appointment.

MySQL Connector driver version: mysql-connector-j-8.0.32
